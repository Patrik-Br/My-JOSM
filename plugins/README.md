*Backup of plugins that I am using plus some I created with Claude AI.

I am also using https://github.com/MissingMaps/3rdPassJOSMPlugin

## Layout

- `MapathonQA/` — source, build scripts, and compiled jar for the MapathonQA plugin (actively developed here).
- `BetterWorkspace/` — source, build scripts, and compiled jar for the BetterWorkspace plugin (actively developed here).
- `other-plugins/` — pre-built jars for everything else, no source in this repo for these. These are
  other authors' plugins kept here as a personal backup; they retain their own original licenses,
  not the GPLv3 terms in `MapathonQA/LICENSE` / `BetterWorkspace/LICENSE`.
